# STACK 

A simple game where you use your **spacebar** to drop blocks and stack them on top of each other to get the tallest stack!

- Try it for yourself! : https://subtomic1833.github.io/Stack/

-------

##  Gameplay

- Press **SPACEBAR** to start the game
- Press **SPACEBAR** to drop the moving block
- Match the blocks as closely as possible
- Each successful stack increases your score
- Missing the stack ends the game
- Press **SPACEBAR** after game over to play again
